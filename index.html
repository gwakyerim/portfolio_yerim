안 녕 하 세 요 !
브랜드의 가치를 최대한으로 뽑아내는 퍼블리셔 곽예림 입니다.

하던 일을 전부 그만두고 새로운 시작에 첫걸음을 한다는 것. 코딩을 접하기 전까진 고민도 많았고 두려웠습니다.
하지만 HTML, CSS, Javascript, jQuery 등 코드가 하나하나 적용되고 완성 되어가는 모습을 볼 때면 행복함이 저를 감싸는 만큼, 이 일이 제 일이라고 확신하고 새로운 발돋움을 시작하게 되었습니다.

저는 "웹 표준"과 "웹 접근성"을 준수하여 사용자가 보기 쉽고 사용하기 편리하게 결과물을 잘 구현합니다. 또한 원활한 협업능력과 피드백을 유연하게 받아들일 수 있는 역량을 가지고 있습니다.

현재 지원 업무는 퍼블리셔이지만,추후 요구하는 역량이 있다면 언제든지 배울 준비도 되어있습니다.

시간내어 읽어주셔서 감사합니다.

SKILLS

  • HTML5 웹 표준 및 웹 접근성을 준수하여 시멘틱 태그의 용도를 정확히 알고 사용할 수 있습니다.
  • flex,grid 를 통한 반응형 사이트 제작과 애니메이션 사용이 가능합니다.SCSS 전처리기 사용이 가능합니다.
  • function 기능을 이용하여 슬라이드 동작과 토글,팝업 등 간단한 이벤트를 사용할 수 있습니다.
  • 클릭 이벤트, 체인지 이벤트 등 여러 이벤트를 사용 가능하며, 애니메이션 구현이 가능합니다.
  • 피그마를 활용한 기획안, 와이어프레임, 프로토타입을 제작할 수 있습니다.
  • git 명령어를 활용한 gitHub 프로젝트 관리가 가능합니다.
  • 소스 코드 편집툴은 VS Code와 Brackets 사용이 가능합니다.
