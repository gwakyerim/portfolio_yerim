안녕하세요! 누구보다 소통과 협업에 진심인 신입 웹 퍼블리셔 곽예림입니다.

하던 일을 전부 그만두고 새로운 시작에 첫걸음을 한다는 것. 코딩을 접하기 전까진 고민도 많았고 두려웠습니다. 하지만 코딩을 알아갈수록 두려움이 용기로 바뀔만큼 흥미롭고 매력 있는 일이라는 것을 깨닫게 되었습니다. 저는 "웹 표준"과 "웹 접근성"을 준수하여 사용자가 보기 쉽고 사용하기 편리하게 결과물을 잘 구현합니다. 또한 팀원들과의 소통을 중요시하며 코딩을 진심으로 재밌어하는 퍼블리셔입니다. 현재 지원 업무는 퍼블리셔이지만, 추후 프론트엔드 개발자의 꿈이 있어 다양한 개발 스택(React,Vue.js)들을 쌓아갈 예정입니다.감사합니다!

SKILLS

  • HTML5 웹 표준 및 웹 접근성을 준수하여 시멘틱 태그의 용도를 정확히 알고 사용할 수 있습니다.
  • flex,grid 를 통한 반응형 사이트 제작과 애니메이션 사용이 가능합니다.SCSS 전처리기 사용이 가능합니다.
  • function 기능을 이용하여 슬라이드 동작과 토글,팝업 등 간단한 이벤트를 사용할 수 있습니다.
  • 클릭 이벤트, 체인지 이벤트 등 여러 이벤트를 사용 가능하며, 애니메이션 구현이 가능합니다.
  • 피그마를 활용한 기획안, 와이어프레임, 프로토타입을 제작할 수 있습니다.
  • git 명령어를 활용한 gitHub 프로젝트 관리가 가능합니다.
  • 소스 코드 편집툴은 VS Code와 Brackets 사용이 가능합니다.
